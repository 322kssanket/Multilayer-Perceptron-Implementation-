 
MLP implementation for 'Wifi Indoor Localization ' dataset.

# For compiling code
Run: g++  mlp.cpp  mlp.h  perceptron.h  perceptron.cpp  layer.cpp  layer.h  main.cpp

# For running the program
Run ./a.out
Output is stored in `mlpWeights.txt` file in the form of weight matrices for each iteration

Description :
Wifi Indoor Localization dataset with 7 input attributes and 4 output classes. It is a multi- class problem.
GUI is in Android.

Link for dataset:-https://archive.ics.uci.edu/ml/datasets/Wireless+Indoor+Localization
